# 💪 哑铃三分化训练手册（家庭版）

> 🎯 目标：减脂 + 保肌肉 + 出线条  
> 🏠 场景：居家训练（哑铃 + 徒手）  
> 📌 特点：高频刺激（比五分化更适合自然训练）  
> 📌 用法：直接照练（循环执行）

---

# 🧠 训练规则
- 每个动作：**3～4组 × 8～15次**
- 组间休息：**60～90秒**
- 重量选择：**最后2次吃力但能完成**
- 训练节奏：**3天循环**

---

# 🔥 热身（必做）
> 每次训练前 5–7 分钟

- 肩部环绕：前后各20次  
- 猫牛式：10次  
- 开合跳：30秒 × 2组  
- 徒手深蹲：15次  

---

# 📅 训练安排（三分化循环）

| 天数 | 内容 |
|------|------|
| Day1 | 推（胸 + 肩前 + 三头） |
| Day2 | 拉（背 + 二头 + 后束） |
| Day3 | 腿 + 核心 |
| Day4 | 休息 / 轻有氧 |
| 然后 | 循环 |

# 🔥 有氧安排（跳绳）

- 推日：10–15分钟  
- 拉日：10–15分钟  
- 腿日：❌ 不跳  
- 休息日：快走或轻跳  

---

# 🧘 拉伸（练后必做）

每个动作20–30秒：

- 胸部拉伸  
- 背部拉伸  
- 大腿前侧  
- 大腿后侧  
- 肩部拉伸  

---

# 🎯 总结

👉 三分化比五分化更适合减脂期  
👉 高频训练 = 更容易保住肌肉  
👉 背和腿优先级最高  
👉 核心训练决定腹肌是否明显  
👉 **坚持 > 花哨动作**
